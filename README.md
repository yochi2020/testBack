## development
1. ติดตั้ง Extentsion Eslint ในVsCode
2. ตั้งค่า Format on Save ในVsCode
3. เพิ่ม คำสั่งในsettings.json ในVsCode  
  "editor.codeActionsOnSave": {
        "source.fixAll": true
    },
    "eslint.validate": [
        "javascript",
        "typescript"
    ],
